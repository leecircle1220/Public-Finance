## About

This repository contains my personal study materials for **Public Finance**.

The notes were written while I was taking the Public Finance course at **National Chengchi University (NCCU)**.

Besides lecture notes, the repository also includes homework assignments, TA session materials, and supplementary exercises prepared for civil service senior examinations.

These materials may be helpful if you are:

* Taking Public Finance
* Reviewing public finance theory
* Preparing for exams
* Preparing for civil service senior examinations
* Looking for additional practice problems and explanations

Please note that this repository is a personal collection of study notes and is not an official course website.

## How to Use These Notes

These notes are intended to be used as a **study aid**.

A recommended way to use this repository is:

* Start with `lecture-note/` to understand the main concepts introduced in class.
* Review the `TA-session/` materials for additional explanations and examples.
* Practice with the assignments in `HW/`.
* Use `crash-course/` as supplementary practice for civil service senior examinations.

These notes work best when used alongside lectures, textbooks, and problem-solving practice.
